# Todo App - Project Specification


## Overview
The Todo App is a lightweight task manager that runs entirely in the browser. Users can add, view, and delete tasks.


## Features
- Add new tasks
- Delete existing tasks
- Persist tasks using localStorage


## Technical Requirements
- HTML for structure
- JavaScript for dynamic behavior
- CSS for styling (optional)
- LocalStorage for data persistence


## User Flow
1. User opens the app.
2. Adds a task using the input field and Add button.
3. The task is added to the list and saved in localStorage.
4. User can delete tasks individually.


## Constraints
- Browser-based only
- No backend
- Simple and responsive UI


## Future Improvements
- Mark tasks as completed
- Edit existing tasks
- Filter tasks (All, Active, Completed)
